## ユーザー(ショップ)情報について

| データ項目             | 例                     | 備考         |
| ---------------------- | ---------------------- | ------------ |
| ショップ識別子         | uixdexx..              |              |
| 名前(表示名)           | ショップ名             |              |
| 名前(ユニーク名)       | shop_789               |              |
| 説明                   | ショップの説明         |              |
| アイコン画像           |                        |              |
| 背景画像               |                        |              |
| webサイト              | https//:llll           |              |
| フォロワー数           | 100                    | (初版は無し) |
| お礼メッセージ         | ありがとうございます。 |              |
| メールアドレス         | shop@gmail.com         |              |
| 登録ウォレットアドレス | 0x00…                  |              |

---

1ユーザーは1ショップとし、ユーザー情報とショップ情報は同一とする。
ただし今後に向けて、schemaとtypeは分けて作成し、それを結合してUserモデルを作成する形をとる。
