| feature 名（＝画面単位） | 画面タイトル              | 機能概要                                                                                                                            |
| ------------------------ | ------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| `creator-dashboard`      | 📊 ダッシュボード         | 売上・アクセス・公開中商品の概要を表示。                                                                                            |
| `product-list`           | 🎨 商品管理画面           | 公開中／下書き／非公開の商品の一覧管理。                                                                                            |
| `product-editor`         | 🧱 商品作成・編集画面     | 新規作成・編集・公開設定をすべて 1 画面で実施（タブ構成：基本情報／価格設定／データアップロード／アダルト設定／公開スケジュール）。 |
| `product-preview`        | 👁 プレビュー確認画面      | 購入ページプレビューやメタ情報確認。                                                                                                |
| `product-publish`        | 🚀 公開実行画面           | 商品を公開状態に変更し販売を開始。                                                                                                  |
| `sales`                  | 💰 売上・決済状況画面     | トークン別・日次・累計の売上を表示。                                                                                                |
| `orders`                 | 📑 注文一覧画面           | 注文 ID、ユーザー、状態、金額、TX 情報を確認。                                                                                      |
| `shop-settings`          | 🏪 ショップ設定画面       | プロフィール／お礼メッセージ／SNS リンク／画像を設定。                                                                              |
| `affiliate-settings`     | 🤝 アフィリエイト設定画面 | 紹介率・リンク発行・成果レポートの管理。                                                                                            |
| `account-settings`       | ⚙️ アカウント設定画面     | ログアウト、パスワード変更、認証情報の管理。                                                                                        |
