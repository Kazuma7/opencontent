| feature 名          | 画面タイトル            | 機能概要                                   |
| ------------------- | ----------------------- | ------------------------------------------ |
| `home`              | 🏠 ホーム画面           | トップページ。新着・人気コンテンツ表示。   |
| `contents-list`     | 🔎 コンテンツ一覧画面   | カテゴリ／タグ／検索で絞り込み表示。       |
| `content-detail`    | 🖼 コンテンツ詳細画面    | タイトル・価格・説明・購入ボタンなど。     |
| `auth-login`        | 🔐 ログイン画面         | メール or SNS ログイン＋ウォレット紐付け。 |
| `purchase-detail`   | 🧾 購入詳細画面         | 支払いトークン選択、注文確定。             |
| `wallet-connect`    | 🔗 別ウォレット接続画面 | MetaMask / WalletConnect 接続。            |
| `payment-verifying` | ⏳ 決済検証中画面       | トランザクション確認・Webhook 待機。       |
| `payment-complete`  | ✅ 支払い完了画面       | 購入成功の完了通知。                       |
| `purchase-history`  | 🧾 購入履歴画面         | 過去の購入一覧を表示。                     |
| `order-detail`      | 📦 注文詳細画面         | 購入詳細・ダウンロードリンク・支払い履歴。 |
| `policy-terms`      | ❓ 利用規約画面         | 規約の閲覧。                               |
| `policy-refund`     | 💬 返金ポリシー画面     | 返金条件の表示。                           |
| `policy-age-check`  | 🔞 年齢確認画面         | 成人向け確認。                             |
| `error-payment`     | ⚠️ 支払いエラー画面     | トランザクション失敗時のエラー表示。       |
